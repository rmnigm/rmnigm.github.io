    Introduction #

    Python is an industry standard for data manipulation, statistical computatuions, machine learning and visualization. It is also extremely easy to start programming with Python, yet mastering this language can be tricky. It offers a wide range of tools and frameworks, most of which are API to complex projects written in C, or C++, or any other language really.

    Basic syntax and programming introduction will not be a part of this course,so if you’re new - I’d recommend to go through any open-source course. I would suggest Yandex’s handbook with around 260 problems, more than enough to get good enough with Python. Overall prerequsites - two semester course of programming in any object-oriented language. You shouldn’t be afraid of classes, file streams, exceptions and basic concepts of memory management, compilation and interpretation difference, stuff like that.
