    CV (Компьютерное зрение) #

    Компьютерное зрение - область, связанная с обработкой фото, видео и любых потоков данных изображений. Та часть, которую я описываю здесь, включает в себя стандартные инструменты манипуляции данными, классические вычислительные методы и нейросетевые подходы для распознавания и обработки объектов.

    CS131 #

    Не самый знаменитый, но крутой курс Стэнфорда по основам компьютерного зрения. Здесь расскажут про основные концепции CV, интерполяцию, сшивание панорам, метрики, задачи и классические подходы с минимальным включением DL.

    CS231N #

    Самый знаменитый курс Стэнфорда по нейросетям для компьютерного зрения. Много теории, много практики по большому разбросу тем - от простеньких kNN до современных vision-трансформеров.
