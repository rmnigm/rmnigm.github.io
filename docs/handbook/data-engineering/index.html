    Дата-инженерия #

    Иногда данных становится так много, что нельзя просто обрабатывать их говнокодом в Jupyter-ноутбуке и таскать SQL-селектами с production-базы. Тогда приходится нанимать дата-инженеров, чтобы они делали отказоустойчивую доставку и обработку нужных данных для всех потребителей. Круто, да?

    SQL #

    Интерактивный учебник попроще #

    SQL Bolt – простенький, но красивый и хороший учебник, чтобы понять основные концепции SQL и научиться основному синтаксису. Must read для любого, кто будет работать с данными – пригодится.

    Задачник посложнее #

    SQL-EX – огромный набор задач, в том числе довольно сложных, по реляционным языкам. Идеально, чтобы оттачивать скиллы к собеседованиям и проверять себя на действительно нетривиальных задачах.
