    Машинное обучение #

    Здесь ссылки на курсы, задачи, лекции, учебники, проекты, open-source инструменты и статьи про машинное обучение. Главный инструмент у большинства аналитиков при проверке гипотез кроме статистических тестов - это, пожалуй, регрессия. А уж что можно сделать с моделями сложнее!

    Учебник ШАД #

    Онлайн-учебник ШАД по классическому машинному обучению, основанный на лекциях К. Воронцова. Он интерактивный, невероятно интересный, покрывает довольно много тем из классического машинного обучения и подходит даже если надо вспоминать теорию перед собеседованиям.

    Записи лекций ШАД #

    Лекции К. Воронцова, одного из самых главных русскоязычных преподавателей машинного обучения. Довольно много математики, разные области (от нейросетей и рекомендательных систем до вероятностных алгоритмов), 30+ лекций с глубоким погружением.

    Курс ML на ФКН ВШЭ #

    Репозиторий курса по классическому машинному обучению от Евгения Соколова, читающийся на ФКН ПМИ. Самое полезное в нем - это огромный набор домашних заданий и записей по теории. Два семестра, но без глубокого обучения, есть ссылки на лекции и семинары.

    Гитхаб ШАДа #

    Профиль Школы анализа данных на гитхабе, где лежат материалы по большому количеству их курсов - глубокое обучение, компьютерное зрение и графика, обработка естественного языка (есть онлайн-учебник!) и мини-курс по беспилотным автомобилям.
